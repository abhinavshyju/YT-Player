$ getting_started

YT-Player is a lightweight JavaScript wrapper for the YouTube Player API that provides a simple and intuitive interface for embedding and controlling YouTube videos in your web applications.

$ features

  • [✓] Easy to use API for YouTube video embedding
  • [✓] Full control over playback (play, pause, seek, volume)
  • [✓] Event handling for player state changes
  • [✓] Responsive design support
  • [✓] TypeScript support included

$ api_methods

play() Starts video playback
pause() Pauses video playback
setVolume(volume) Sets the volume (0-100)
seekTo(seconds) Seeks to a specific time
